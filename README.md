# IMS Wordpress plugin

this is a plugin for managing interns from a telegram bot and wordpress admin dashboard.

---

Interns Can:
- Start the bot from Telegram
- Click the "Start Work" button for setting the start time.
- Click "End Work" after finishing the job session.
- Send daily logs using "Send daily log" button

---

Admins can:
- View the users, manage and accept new user requests
- See Overall Statistics in dashboard (User count, Active users, View sent daily logs, and latest activities)
- View all of the daily logs with filtering options and exporting as csv, pdf, Excel.
- View statistics of a user (joined date, today's total time worked, job activity, Telegram information)
- Set admins in telegram (including themselves) for daily log notification or pending users.
